# Record Notification 组件更新通知

#### 默认配置如下组件通知功能

- "@choicefe/scm-widgets"
- "@choicefe/scm-permission"
- "@choicefe/scm-components"
- "@choicefe/scm-modal"
- "@choicefe/scm-utils"
- "@choicefe/scm-common"
- "@choicefe/scm-constant"

#### 支持自定义配置

- 1. 默认 60 分钟同步一次

- 2. 扩展组件配置

```json
{
  "cook.component.names": ["@choicefe/picker", "@choicefe/select"]
}
```
