# Record Notification 组件更新通知

#### 自定义配置

- 1. 同步时间, 默认 60 分钟一次

- 2. 扩展组件配置

```json
{
  "cook.scope": "@choicefe", // 私域@choicefe下的所有组件
  "cook.components": ["picker", "select"] // 其他不在私域下的组件, 与私域是"或"关系
}
```
